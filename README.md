# Multiplayer Snake Game

A real-time multiplayer snake game with authentication, game logging, and spectator mode.

## Features
- **Multiplayer**: Play with friends in real-time
- **Authentication**: Sign in/Sign up with Firebase
- **Game History**: Track your plays, wins, and statistics
- **Leaderboard**: See top players
- **Room System**: Create/join game rooms

## Quick Start

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
pip install -r requirements.txt
python main.py
```

## Tech Stack

**Backend:** Node.js, Express, Socket.IO, Firebase  
**Frontend:** Python, Pygame, Socket.IO Client
